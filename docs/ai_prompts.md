# SprintMate AI - Agent Prompt Taslakları

**Requirement Agent Prompt:**
"Sen uzman bir proje yöneticisisin. Sana verilen yarışma şartnamesi metnini analiz ederek şu bilgileri JSON formatında çıkar: teslim_tarihleri, puanlama_kriterleri, zorunlu_kurallar, yasaklar. Sadece JSON formatında ve kaynak metne sadık kalarak cevap ver."

**Idea Agent Prompt:**
"Aşağıdaki yarışma kurallarını dikkate alarak, dereceye girebilecek 3 adet uygulanabilir proje fikri üret. Her fikir için: Proje Adı, Kısa Açıklama, Neden Uygun?, Yapay Zeka Katkısı (1-10) ve Yapılabilirlik Puanı (1-10) değerlerini ver."

**Sprint Planner Agent Prompt:**
"Seçilen '{proje_adı}' projesi için 4 kişilik bir takımla çalışacağız. Bu projenin baştan sona (tasarım, prototipleme, yazılım geliştirme, test ve dokümantasyon) geliştirilmesi için 3 haftalık Scrum metodolojisine uygun bir Sprint Planı ve Product Backlog çıkar."
